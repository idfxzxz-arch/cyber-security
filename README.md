# WD Cybersecurity Awareness Portal

Portal edukasi keamanan siber untuk karyawan WD Group Company. Dibuat dengan React JS, Vite, Tailwind CSS, React Router DOM, Lucide React, dan data dummy JavaScript.

## Fitur

- Halaman beranda edukasi
- Daftar materi cybersecurity
- Detail materi
- Simulasi keamanan siber
- Simulasi praktik berbasis skenario
- Tips keamanan
- Kuis ringan dengan skor
- Halaman tentang

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
